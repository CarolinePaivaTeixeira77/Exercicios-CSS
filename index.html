Seletores de Texto em CSS

Seletores de texto em CSS permitem estilizar partes específicas de um documento HTML de forma detalhada e personalizada. Com eles, podemos definir estilos para textos de maneira segmentada, aplicando propriedades como font-size, color, text-decoration e muito mais.

Principais Tipos de Seletores

  • Seletores de Elemento: estilizam todos os elementos de um tipo específico, como p, h1, etc.
  • Seletores de Classe: aplicam estilos a elementos com uma classe específica, usando o prefixo . (ex: .minha-classe).
  • Seletores de ID: aplicam estilos a elementos com um ID único, usando o prefixo # (ex: #meu-id).
  • Seletores de Atributo: permitem selecionar elementos com um atributo específico (ex: [type="text"]).
  • Pseudo-classes: aplicam estilos em estados específicos de um elemento, como :hover e :focus.

Exemplos

Veja alguns exemplos de seletores de texto em CSS:

          
            /* Seleciona todos os parágrafos */
            p {
              color: blue;
            }
      
            /* Seleciona elementos com a classe "destaque" */
            .destaque {
              font-weight: bold;
            }
      
            /* Seleciona o elemento com ID "titulo" */
            #titulo {
              font-size: 24px;
            }
